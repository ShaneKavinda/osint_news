# Real Time News Analyzer

## 1. Goal

Demonstrate how a real time news feed could be used to identify trends, patterns and visualize such information graphically for easy interpretation

## 2. Project setup

- Create a python virtual environment using `venv` or Anaconda (This project uses python 3.14)
````bash
python -m venv venv
./venv/Scripts/activate
````

- Install the dependencies in the `requirements.txt` file
````bash
pip install -r requirements.txt
````

## 3. Running the project

- Inject news articles from different RSS feeds

```bash
python inject.py
````
